# gophersfrm.netlify.com

## Developing

1. Install [GoHugo](https://gohugo.io/getting-started/quick-start/#step-1-install-hugo)
2. initalize submodules

    ```shell
    git submodule update --init --recursive
    ```

3. Start GoHugo

    ```shell
    hugo server -D
    ```
